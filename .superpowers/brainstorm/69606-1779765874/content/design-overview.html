重要决策
✓Tailwind v4（CSS-first config，无 tailwind.config.js）
✓shadcn 暗色主题 CSS 变量完全替换 global.css 现有 token
✓WorkspacePage 三栏布局保留 grid，只换样式写法
✓packages/ui/ 现有组件暂不动（前端未使用），迁移后可统一重写
⚠CSS 动画（pulse/spin）迁移到 Tailwind animate-pulse / animate-spin
⚠onFocus/onBlur 内联样式切换 → focus-visible: Tailwind ring utilities
